# FPS Benchmark — WebGL Browser Engine

A modern, light, dark-themed static website for benchmarking client-side browser rendering performance and calculating overall FPS score.

## Features
- **30-Second Auto Test Run**: Stress tests frame rates continuously.
- **Pure JavaScript & Canvas**: Zero third-party runtime dependencies.
- **Real-Time Graph**: Dynamic visual framerate monitor.
- **Hardware Sniffer**: Auto-detect GPU, CPU logical core count, DPR, and device resolution.
- **Local History**: Saves benchmark histories in client LocalStorage.

## GitHub Pages Deployment
1. Push this repository to GitHub.
2. Navigate to **Settings** > **Pages**.
3. Under **Build and deployment**, select `main` branch and `/ (root)`.
4. Save and launch your live benchmark URL instantly!
