About FPS Benchmark

FPS Benchmark renders a procedural, multi-layered raymarching-inspired 2D fragment shader on a Canvas element. By continuously shifting pixel values across high frame rates, it stresses both single-core CPU draw execution and GPU rasterization pipelines to deliver an accurate baseline metric of modern browser hardware acceleration performance.
